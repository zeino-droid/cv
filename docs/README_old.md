# 🎓 Job Copilot — Ingénieur R&D

> **Objectif** : Décrocher un CDI d'ingénieur en Simulation ou Énergie (France) avant fin août 2026.
> **Stratégie** : Automatisation intelligente des candidatures et adaptation fine des CV.

## 🚀 Démarrage Rapide

1. **Mass Apply (CLI)** : La méthode la plus efficace pour envoyer 50 candidatures/semaine.
   ```bash
   python fast_apply.py --scan --min-score 50
   ```
2. **Dashboard (UI)** : Pour visualiser le marché et suivre tes candidatures.
   ```bash
   streamlit run app.py
   ```

## 🏗 Structure du Projet

- `core/` : Le cerveau (matching, LLM rapide, rendering Typst).
- `config/static_cvs/` : Tes 2 profils de base (Simulation R&D vs Énergie/CFD).
- `data/jobs.db` : Base de données SQLite unique (Tracking + Offres).
- `output/` : Tous les CV adaptés et les lettres de motivation générés.
- `docs/` : Stratégie de convergence, revue de code, et historique.

## 📄 Les 2 CV Maîtres

Nous avons "figé" deux versions de ton CV pour maximiser tes chances :
1. **Simulation R&D** : Focus sur Abaqus, Metafor, Éléments finis, Flambage dynamique (PFE).
2. **Énergie & CFD** : Focus sur Jumeaux numériques, Audit énergétique, Ansys Fluent, Décarbonation.
